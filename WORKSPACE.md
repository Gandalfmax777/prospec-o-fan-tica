# Workspace (Frontend + Backend)

Este repositório é o **frontend** do projeto "CDR de prospecção".

O backend está no repositório/pasta: `prospec-o-fan-tica-backend/`.

## Rodando local

1. Backend em `http://localhost:3333`
2. Frontend:

- copie `.env.example` para `.env`
- `npm install`
- `npm run dev`
- abre `http://localhost:5173`

## Dependências importantes

- React + TypeScript + Vite
- UI: shadcn/ui (Radix)
- Charts: recharts
- Auth: Better Auth (sessão via cookie)
